Shopping List

Create a shopping list in the variable myList. The list should be a multi-dimensional array containing several sub-arrays.

The first element in each sub-array should contain a string with the name of the item. The second element should be a number representing the quantity i.e.

["Chocolate Bar", 15]

There should be at least 5 sub-arrays in the list.

Tests

Passed: myList should be an array.
Passed: The first elements in each of your sub-arrays should all be strings.
Passed: The second elements in each of your sub-arrays should all be numbers.
Passed: You should have at least 5 items in your list.

// code
const myList = [
  ["chocolate", 15],
  ["banana", 10],
  ["cery", 20],
  ["meatball", 20],
  ["apple", 10]
];
